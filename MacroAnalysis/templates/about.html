MacroAnalysis

MacroAnalysis is a CS50 final project designed to allow users to easily see and compare almost every major macroeconomic indicator for nearly every country in the world. All data is directly from the World Bank. Forecasts (currently disable because of resource constraints) are from the IMF.

Key Features

  • 🌍 Interactive world map for country selection 🌍
  • 📈 Dynamic line graphs 📈
  • ⚖️ Support for comparing multiple indicators and countries ⚖️
  • 🌑 Dark-themed UI, as that should be the default 🌑
  • 📡 Real-time economic data 📡
